¿Qué es Argentina Programa?

Argentina Programa: Es una capacitación en programación gratuita y virtual organizada en dos etapas:

1- #SéProgramar, en donde vas a aprender los fundamentos básicos para aprender a programar.

2- #YoProgramo, en la que vas a profundizar tus conocimientos y capacitarte en nueve modulos para convertirte en un programador web full stack junior.

Las dos etapas cuentan con una certificación conjunta del Ministerio de Desarrollo Productivo y la Cámara Argentina de la Industria de Software (CESSI). La segunda etapa tiene además la certificación del Instituto Nacional de Tecnología Industrial (INTI).

Módulo 1

En este módulo aprenderás lo referido a protocolos en la red ejemplo htpp, y como se relacionan entre cliente/ servidor. Además aprenderás sobre la tecnología Scrum, la cual se aplica en los equipos de software para ser más productivos. También aprenderás a gestionar tu tiempo.

Módulo 2

Aquí aprenderás a usar HTML: Lenguaje de etiquetas de hipertexto. Aprenderás Css para dar estilos junto al lenguaje Javascript para darle algunas funciones a los elementos. Al finalizar este modulo aprenderás sobre TypeScript el cual es un lenguaje que ayuda a resolver algunos problemas de javascript

Módulo 3

En este módulo aprenderás sobre Angular. Este es un framework de ingeniería de software utilizado para crear aplicaciones web de una sola página. Este framework que vincula JavaScript y HTML, el código entre ellos se sincroniza. Este mecanismo reduce la cantidad de código para escribir.

Módulo 4

Aquí aprenderás sobre " Base de Datos " la cual es una recopilación organizada de información o datos estructurados, que normalmente se almacena en un sistema informático. También vas a poder gestionarlas e implementarlas utilizando la tegnología (DBMS) sistema de gestión de bases de datos.

Módulo 5

En este módulo aprenderás sobre Programación Orientada a Objetos, Fundamentos de POO, Análisis y Diseño Orientado a Objetos, Design Patterns. Esta parte es muy importante ya que al aprender esto podremos continuar llevando todo lo aprendido a un lenguaje de Programación Orientada a Objetos.

Módulo 6

Aquí aprenderás sobre Java: Esta es una tecnología que se usa para el desarrollo de aplicaciones que convierten a la Web en un elemento más interesante y útil colocando en practica toda la teoria de el modulo anterior. También aprenderás sobre Estructuras Estáticas - Arreglos (Arrays), Colecciones, Excepciones.

Módulo 7

En este módulo seguirás aprendiendo sobre java, sin embargo ahora lo llevaremos a la web utilizando Java Web. También aprenderemos sobre JSP (Java Server Pages), Servlets, Sockets. Estas herramientas nos ayudarán a tener un sitio web dinamico cada vez mas completo, estructrado, funcional y eficiente.

Módulo 8

Aquí aprenderás sobre un Framework llamado Spring Boot, API Con Spring Boot, Probando la API, Aplicando buenas prácticas de diseño, Práctica - Creando estas APIs básicas, Práctica - Con patrones de diseño, y al finalizar desarrollaremos Práctica API con Spring Boot (JPA + HIBERNATE), para web completa

Módulo 9

En este módulo aprenderás sobre Seguridad Web para saber como proteger tus sitios webs y como prevenir de ser bulnerable, Testing para realizar pruebas en cada paso de tu sitio o tu proyecto, también aprenderás sobre Deployment, y comunicacián afectiva para formarte como uno de los mejores comunicadores. .
